Try it!

Upload an image to find out whether or not your item is recyclable!

Revitacycle is a revolutionary new website in the works which allows users to upload a picture of a certain object and figure out whether the object is recyclable or not. We are using machine learning tools to compare a user imported image to a general database. We then use the data collected from the image to figure out the top items it could be. The items are then filtered recyclable or not and then when the result is computed, the user will see an output of “Your item is recyclable” or “Your item is not recyclable”.
